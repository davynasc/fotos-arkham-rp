# FiveM Image Host

Este repositório é usado para hospedar imagens que serão utilizadas em um servidor de GTA RP FiveM.

Para usar uma imagem, você deve obter o link "raw" (cru) do arquivo.

**Como obter o link raw:**

1.  Navegue até a imagem desejada no repositório.
2.  Clique no botão "Raw".
3.  Copie o URL da barra de endereços do seu navegador.

O link terá o formato: `https://raw.githubusercontent.com/SEU_USUARIO/fivem-image-host/main/caminho/para/sua/imagem.png`

**Exemplo de uso no FiveM:**

Se o servidor utiliza um comando ou script para exibir imagens (como em celulares ou painéis), você deve usar o link raw completo.

*Lembre-se: O GitHub é um serviço de hospedagem de código. Para garantir a estabilidade e o bom funcionamento, evite fazer upload de arquivos muito grandes ou em grande quantidade.*
